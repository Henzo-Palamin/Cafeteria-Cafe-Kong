Café Kong is my first complete web development project, created to put into practice the knowledge I have gained in HTML5 and CSS3.

The goal of this project was to design and develop a modern, responsive landing page for a fictional coffee shop, focusing on clean design, intuitive navigation, and an enjoyable user experience.

Features:
Responsive layout
Coffee and food menu sections
Image gallery
Location section
Contact section
Custom visual identity

Technologies Used:
HTML5 & CSS3

What I Learned

Throughout this project, I strengthened my understanding of:

Semantic HTML structure
CSS styling and layout techniques
Flexbox for responsive positioning
Responsive web design for different screen sizes
Code organization and front-end best practices

About This Project

This project represents the beginning of my journey as a Front-End Developer. It allowed me to transform what I had learned into a complete and functional website while gaining valuable hands-on experience in web development.

As I continue learning new technologies and improving my skills, I plan to build more projects that reflect my growth as a developer.
